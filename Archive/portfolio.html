My Portfolio

Programming: Heaven Jumper

Inspired by Ori and the Blind Forest, this jump-centered game is, for now, my crowning achievement. Made alongside my friend, Philippe Lizotte, this game has sprites, level design, and general feel that demonstrates my current skill with creating video games.

Programming: Cube

Inspired by the 1997 movie of the same name, this game involves a player entering through different rooms, gathering artifacts, while avoiding the deadly traps that lay ahead. This was my first game ever made using Javascript Canvas.

Programming: Refurbished Immortality

My most recent project, done entirely in JavaScript, is an interactive ARG involving analog horror akin to the Mandela Catalogue and Walten Files. Done within a few months, it is still actively being worked on.

Animation: Climate's Graveyard

This was my final project for my college Animation class, and my first big piece of animation.

Film: Detracted Minds

Similar to Animation, I had done 2 semesters of Filmmaking, and this was my final assignment for Filmmaking 2. With the help of Lucciano Vezina, along with many others, we were able to get together an 8-minute-long short movie.

Film: Panic at Work

A more in-depth film project, Panic At Work is a final project for a Job Preparation class about what not to do in the workplace. With the help of Artimes Fallah and starring Carmina-Ioana Rotaru, Palmarinao DiMarco and Kevin Rumboldt, this 24-minute long short film, a much bigger project than the afformentioned, describes my current filmmaking capabilities.
